Description:

An online hotel booking website that allows customers to search rooms by date, type, and price; book and cancel reservations; track booking history; and leave hotel reviews. The admin panel supports hotel, branch, room, booking, user, comment, and report management. Built as a monolithic ASP.NET Core MVC project with a 3-layer architecture.

Contributions:
  • Designed system architecture and database schema with EF Core Code-First.
  • Built CRUD for hotels, branches, rooms, users, and admins using ASP.NET Core MVC + EF Core.
  • Implemented booking workflows with secure session handling and role-based authorization.
  • Applied Repository Pattern, Fluent API, AutoMapper for cleaner data management.
  • Integrated AdminLTE dashboard with pagination, statistics, and notifications.
